General Fields:

background_color: Specifies a background color for the NFT's display.
category: Categorizes the NFT (e.g., Art, Music, Collectibles).
explicit_content: Flags the NFT as containing explicit or sensitive content.
license: Indicates the licensing terms for the NFT content.
media: (Alternative to "image" or "animation_url") A general field for various media types.
subtitle: Provides a secondary descriptive text for the NFT.
Platform-Specific Fields:

chain: Specifies the blockchain network on which the NFT is minted (e.g., Ethereum, Polygon).
token_id: Identifies the unique token ID of the NFT within its collection.
token_standard: Indicates the NFT standard used (e.g., ERC-721, ERC-1155).
symbol: Specifies a symbol for the NFT collection.
seller_fee_basis_points: (OpenSea) Sets a custom fee percentage for the seller.
is_presale: (OpenSea) Indicates if the NFT is part of a presale.
start_time: (OpenSea) Sets a start time for the NFT's sale.
end_time: (OpenSea) Sets an end time for the NFT's sale.
tags: (Rarible) Adds keywords for search and discovery.
Game-Related Fields:

stats: Stores numerical stats or attributes for game items or characters.
levels: Indicates progression levels or tiers within a game.
abilities: Lists special abilities or skills associated with a game character or item.
